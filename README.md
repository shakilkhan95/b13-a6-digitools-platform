# 🛠️ DigiTools — Digital Tools Buying Website

A modern, fully responsive digital tools marketplace built with React, Tailwind CSS, and DaisyUI. Browse, add, and manage premium digital products with a smooth cart experience and toast notifications.

## 🧰 Technologies Used

| Technology          | Purpose                   |
| ------------------- | ------------------------- |
| **React.js**        | Component-based UI        |
| **Vite**            | Lightning-fast build tool |
| **Tailwind CSS**    | Utility-first styling     |
| **DaisyUI**         | Pre-built UI components   |
| **React-Toastify**  | Toast notifications       |
| **JavaScript ES6+** | App logic                 |
| **JSON**            | Product data source       |

---

## ✨ Features

1. **🛒 Interactive Cart System** — Add products to cart, view total price, remove individual items, or clear everything with Proceed to Checkout. Cart count updates live in the navbar.

2. **🔔 Toast Notifications** — Every cart action (add, remove, checkout) triggers a styled toast notification via React-Toastify, giving users clear and instant feedback.

3. **📱 Fully Responsive Design** — Optimized for all screen sizes from mobile to 1600px desktop, with a collapsible mobile navbar and adaptive grid layouts.
